# PulseDock v0.1.1

PulseDock v0.1.1 is the first GitHub Releases build for Windows.

## Highlights

- tray-first desktop app for local AI coding usage snapshots
- Codex usage support from local session files
- Cursor usage support from local desktop auth and export data
- provider detail views and manual refresh

## Install

- Download `PulseDock-Setup-0.1.1.exe` from this release.
- Run the installer on Windows.
- If Windows SmartScreen appears, use `More info` then `Run anyway` if you trust the release source.

## Known Limitations

- Windows only
- unsigned installer for this first public release
- manual reinstall required for updates
